# Kheiron-UI (KUI)

### **pnpm**

```bash
npm install -g pnpm
```

### **Install**

```bash
pnpm install
```

### **Storybook**

Run proyect with storybook, usign this command:

```bash
pnpm storybook
```

After open the documentation in [http://localhost:6006](http://localhost:6006)

### **ESLint**

```
pnpm lint
```

### **Formater**

```
pnpm prettier
```

## Recommendation

_Go to the setting_

> File > Preferences > Settings

Search _"Format On Save"_ and checked the editor option.
