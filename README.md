# Kheiron-UI (KUI)

### **pnpm**

```bash
npm install -g pnpm
```

### **Install**

```bash
pnpm install
```

### **Storybook**

After open the documentation in [http://localhost:6006](http://localhost:6006)

```bash
pnpm storybook
```
